Dog API Fetch Project

This project demonstrates how to use the JavaScript Fetch API to retrieve and display data from the Dog API.

Features
- Fetch list of dog breeds
- Display breed names
- Click a breed to view details
- Fetch random dog facts
- Display dog groups
- Error handling for API requests

How to Run
1. Download or clone the repository
2. Open index.html in a browser
3. The application will load dog breeds automatically

Technologies Used
- HTML
- JavaScript
- Fetch API
- Dog API
