
# React Frameworks Exploration Assignment

This project demonstrates the use of **different React frameworks** by building small applications with each.
The goal is to explore their features, workflows, and use-cases.


react-frameworks-assignment/
│
├── next-app/ # Next.js landing page with dynamic user pages
├── remix-app/ # React Router blog with dynamic post routing
├── gatsby-app/ # Gatsby portfolio site
├── expo-app/ # Expo mobile app with Home and About tabs
├── README.md # This file
└── .gitignore # To ignore unnecessary files like node_modules
## How to Run

React App:
cd react-app
npm install
npm run dev

Next.js App:
cd next-app
npm install
npm run dev

Gatsby App:
cd gatsby-app
npm install
gatsby develop

Expo App:
cd expo-app
npm install
npx expo start
