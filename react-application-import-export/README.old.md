# React Application Import Export

This project demonstrates component modularity in React using default and named imports and exports.

## Components
- Header (default export)
- Footer (default export)
- ContentA (named export)
- ContentB (named export)
- Button (shared named export)

## How to Run

1. Clone the repository
2. Run npm install
3. Run npm start

## Features
- Modular React components
- Shared reusable button component
- Demonstrates default and named imports/exports
